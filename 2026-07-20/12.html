AI-Generated Architecture Diagrams in VS Code Using Draw.io MCP

Manoj Saini · Cloud Native Daily · 7 min · 원문 보기 (Medium)

요약

Draw.io MCP 서버를 VS Code에 연결하면 자연어로 인프라를 설명하는 것만으로 편집 가능한 아키텍처 다이어그램이 바로 만들어진다.

저자는 AI 코딩 어시스턴트가 이제 단순 코드 자동완성을 넘어 다이어그램 같은 문서 작업까지 함께 처리하는 파트너가 되어가고 있다고 말한다. Draw.io MCP 서버를 VS Code에 붙이면 인프라 구성을 말로 설명하는 것만으로 AI가 draw.io XML을 생성해 아키텍처 다이어그램을 그려준다. 기존처럼 이미지로 내보내거나 브라우저와 IDE를 오가며 그림을 다시 그릴 필요 없이, IDE 안에서 바로 결과물을 확인하고 수정할 수 있다는 점이 핵심 장점이다. 글은 이 MCP 서버를 설치하고 VS Code 환경을 설정하는 과정을 단계별로 안내하며, 로드밸런서·EC2·RDS로 구성된 3계층 AWS 아키텍처 같은 실제 예시로 활용법을 보여준다. 결과물이 고정된 이미지가 아니라 언제든 수정 가능한 파일로 나온다는 점에서 실무 문서화 워크플로우를 크게 단순화해준다.
