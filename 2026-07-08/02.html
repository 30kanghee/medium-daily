Interviewer: In a microservice architecture, how do you solve the problem of join queries?

Umesh Kumar Yadav · 6분 읽기 · 원문

마이크로서비스 환경에서 여러 서비스에 흩어진 데이터를 조인해야 하는 상황을 면접 문답 형식으로 풀어낸 글.

요약

이 글은 실제 기술 면접에서 자주 나오는 "마이크로서비스 아키텍처에서 조인 쿼리 문제를 어떻게 해결하나요?"라는 질문을 다룬다. 저자는 API 조합(API Composition), CQRS를 통한 조회 전용 비정규화 테이블, 데이터 복제, 이벤트 기반 아키텍처 등 여러 접근법을 비교하며 각각의 장단점을 설명한다. 단일 데이터베이스에서는 당연했던 조인이 서비스가 분리되는 순간 왜 어려운 문제가 되는지, 실무에서 어떤 기준으로 방식을 선택해야 하는지를 면접 답변 형태로 정리한 것이 이 글의 핵심이다.
