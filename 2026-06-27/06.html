These 15 Python Problems Show Up In Almost Every Data Engineering Interview

Ankur Gupta · 5분 읽기 · 원문 · freedium

요약

데이터 엔지니어링 면접에서 거의 빠짐없이 등장하는 Python 코딩 문제 15가지를 풀이 코드와 함께 정리한다.

Python은 데이터 파이프라인 구축, 대용량 데이터셋 처리, ETL 작업 작성, 데이터 워크플로 자동화 등 데이터 엔지니어링 전반에 걸쳐 가장 널리 사용되는 언어다. 면접 문제들은 대체로 실무 중심이며, 중복 처리·문자열 처리·집계·변환 등 현실적인 데이터 문제에 집중한다. 이 글은 리스트 조작, 딕셔너리 활용, 알고리즘적 사고를 다루는 15가지 문제와 해법을 제시하며, SQL·PySpark 등과 함께 현대 데이터 엔지니어링 역할의 핵심 기반을 이룬다고 결론 짓는다.

Python은 데이터 엔지니어링에서 가장 널리 쓰이는 언어 중 하나다. 데이터 파이프라인 구축, 대용량 데이터셋 처리, ETL 작업 작성, 데이터 워크플로 자동화에 두루 활용된다.

그래서 많은 데이터 엔지니어링 면접에 Python 코딩 문제가 포함된다. 문제 해결 능력, 데이터 조작 실력, 일반적인 프로그래밍 패턴에 대한 친숙도를 평가하기 위해서다. 이런 문제들은 보통 실무적이며, 중복 처리·문자열 처리·변환·집계처럼 현실적인 데이터 과제에 초점을 맞춘다.

이 글에서는 데이터 엔지니어링 면접에서 자주 출제되는 Python 코딩 문제 15가지를 풀이 코드와 함께 살펴본다.

1. 리스트에서 중복 요소 찾기

문제

정수 리스트가 주어졌을 때, 중복된 요소를 모두 반환하라.

풀이

nums = [1, 2, 3, 4, 2, 5, 3, 6]
duplicates = list(set([x for x in nums if nums.count(x) > 1]))
print(duplicates)

리스트 컴프리헨션으로 두 번 이상 등장하는 요소만 추린 뒤, set으로 중복을 제거해 고유한 중복 값만 남긴다.

2. 리스트 요소의 등장 횟수 세기

문제

리스트에서 각 요소가 몇 번 등장하는지 세어라.

풀이

nums = [1, 2, 2, 3, 3, 3, 4]
freq = {}
for n in nums:
    freq[n] = freq.get(n, 0) + 1
print(freq)

딕셔너리의 get 메서드를 활용해 키가 없을 때 0을 기본값으로 설정하면서 카운트를 누적한다. 결과적으로 각 숫자와 그 등장 횟수가 매핑된다.

3. 두 번째로 큰 수 찾기

문제

리스트에서 두 번째로 큰 수를 찾아라.

풀이

nums = [10, 20, 4, 45, 99]
nums = list(set(nums))
nums.sort()
print(nums[-2])

먼저 set으로 중복을 제거한 뒤 정렬하면, 뒤에서 두 번째 인덱스(-2)가 두 번째로 큰 값이 된다.

4. 문자열 뒤집기

문제

주어진 문자열을 뒤집어라.

풀이

text = "dataengineering"
reversed_text = text[::-1]
print(reversed_text)

Python의 슬라이싱 문법에서 [::-1]은 시퀀스를 역순으로 순회한다. 가장 간결하고 관용적인 문자열 뒤집기 방법이다.

5. 팰린드롬 판별

문제

주어진 문자열이 앞으로 읽으나 뒤로 읽으나 같은지 확인하라.

풀이

word = "level"
if word == word[::-1]:
    print("Palindrome")
else:
    print("Not Palindrome")

원본 문자열과 뒤집은 문자열을 직접 비교한다. 문자열을 뒤집은 결과가 원본과 같다면 팰린드롬이다.

6. 리스트에서 중복 제거

문제

리스트에서 중복된 값을 모두 제거하라.

풀이

nums = [1, 2, 2, 3, 4, 4, 5]
unique = list(set(nums))
print(unique)

set은 고유한 값만 허용하는 자료구조다. 리스트를 set으로 변환하면 중복이 자동으로 제거되고, 다시 list로 변환해 원하는 형태를 얻는다.

7. 중첩 리스트 평탄화

문제

중첩된 리스트를 평탄한 1차원 리스트로 변환하라.

풀이

nested = [[1, 2], [3, 4], [5, 6]]
flat = [item for sublist in nested for item in sublist]
print(flat)

이중 리스트 컴프리헨션을 사용한다. 바깥 루프가 각 하위 리스트를 순회하고, 안쪽 루프가 그 안의 개별 항목을 펼친다. 결과는 모든 요소가 단일 레벨로 합쳐진 리스트다.

8. 두 리스트의 공통 요소 찾기

문제

두 리스트에 모두 존재하는 공통 요소를 반환하라.

풀이

list1 = [1, 2, 3, 4]
list2 = [3, 4, 5, 6]
common = list(set(list1) & set(list2))
print(common)

두 리스트를 각각 set으로 변환한 뒤 & 연산자(교집합)를 적용한다. 집합 연산이 두 컬렉션에 모두 존재하는 요소를 효율적으로 찾아준다.

9. 문장에서 단어 빈도 세기

문제

주어진 문장에서 각 단어의 등장 횟수를 세어라.

풀이

text = "data engineering is fun and data engineering is powerful"
words = text.split()
freq = {}
for word in words:
    freq[word] = freq.get(word, 0) + 1
print(freq)

split()으로 문장을 공백 기준으로 나눈 뒤, 딕셔너리로 각 단어의 빈도를 누적한다. 2번 문제와 같은 패턴으로, 텍스트 데이터 분석의 기본 기법이다.

10. 리스트에서 가장 큰 요소 찾기

문제

리스트에서 가장 큰 수를 반환하라.

풀이

nums = [10, 5, 25, 8]
largest = max(nums)
print(largest)

Python 내장 함수 max()는 이터러블에서 가장 큰 값을 즉시 반환한다. 반복문이나 정렬 없이 한 줄로 해결된다.

11. 두 딕셔너리 병합

문제

두 딕셔너리를 하나로 합쳐라.

풀이

dict1 = {"a": 1, "b": 2}
dict2 = {"c": 3, "d": 4}
merged = {**dict1, **dict2}
print(merged)

** 언패킹 연산자를 사용해 두 딕셔너리의 키-값 쌍을 새 딕셔너리 리터럴 안에 펼친다. Python 3.5 이상에서 사용 가능한 가장 현대적인 병합 방식이다. 같은 키가 있다면 오른쪽 딕셔너리의 값이 덮어쓴다.

12. 값에 따라 요소 그룹화

문제

동일한 값끼리 묶어서 그룹화하라.

풀이

nums = [1, 2, 2, 3, 3, 3]
group = {}
for n in nums:
    group.setdefault(n, []).append(n)
print(group)

setdefault는 키가 없으면 지정한 기본값(여기서는 빈 리스트)을 설정하고 반환한다. 각 숫자를 자신의 값을 키로 하는 리스트에 추가하면, 같은 값끼리 묶인 딕셔너리가 만들어진다.

13. 연속 수열에서 누락된 수 찾기

문제

1부터 n까지의 수열에서 빠진 숫자를 찾아라.

풀이

nums = [1, 2, 3, 5]
n = len(nums) + 1
missing = n * (n + 1) // 2 - sum(nums)
print(missing)

가우스의 등차수열 합 공식을 활용한다. 1부터 n까지의 합은 n * (n + 1) / 2다. 이 이론값에서 실제 리스트의 합을 빼면 빠진 숫자가 남는다. 정렬이나 집합 연산 없이 O(n) 시간에 해결된다.

14. 딕셔너리 리스트를 값 기준으로 정렬

문제

딕셔너리들의 리스트를 나이(age) 기준으로 정렬하라.

풀이

data = [
    {"name": "Alice",   "age": 30},
    {"name": "Bob",     "age": 25},
    {"name": "Charlie", "age": 35}
]
sorted_data = sorted(data, key=lambda x: x["age"])
print(sorted_data)

sorted()의 key 파라미터에 람다 함수를 전달해 정렬 기준이 되는 값을 지정한다. 각 딕셔너리에서 "age" 값을 꺼내 오름차순으로 정렬한다. 내림차순이 필요하면 reverse=True를 추가한다.

15. 상위 N개의 최댓값 찾기

문제

리스트에서 가장 큰 수 상위 3개를 반환하라.

풀이

nums = [10, 4, 3, 50, 23, 90]
top3 = sorted(nums, reverse=True)[:3]
print(top3)

reverse=True로 내림차순 정렬한 뒤 슬라이싱으로 앞에서 3개를 가져온다. 간결하고 읽기 쉬우며, 상위 N개 패턴에 바로 적용할 수 있다. 대용량 데이터라면 heapq.nlargest(3, nums)가 더 효율적이다.

탄탄한 Python 실력은 SQL, PySpark 같은 분산 처리 프레임워크와 함께 현대 데이터 엔지니어링 역할의 핵심 기반을 이룬다.
